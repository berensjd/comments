# Demos the comments API [JSONPlacholder](https://jsonplaceholder.typicode.com/)

## Available Scripts

In the project directory, you can run:

### Development build

```javascript
$ yarn start
```

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

In development the call to the API is delayed to demo the loading status within each comment box placeholder.. The variable within
.env.development has been set to "true" to trigger this delay

### Production build

```javascript
$ yarn build
```

Builds the app for production to the `build` folder
